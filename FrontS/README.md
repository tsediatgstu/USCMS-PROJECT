# USCMS - University Student Complaint Management System

### Developed by: **Tsedeke Tgstu**

## 📌 Project Overview
USCMS is a full-stack (MERN) responsive web application designed to streamline the communication between students and university authorities. 

## 🚀 Key Features
- **Admin Authority Portal:** Access using `Admin` / `123@`.
- **Student Dashboard:** Submit, track, and view complaint history.
- **Security:** OTP-based password recovery with a 5-attempt lockdown mechanism.
- **Modern UI:** Built with **Tailwind CSS v4** and **Dark Mode** support.
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind v4, Axios, Lucide Icons.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB.

## 📥 Installation
1. Clone the repository.
2. Navigate to `FrontS` and run `npm install`.
3. Create a `.env` file and set `VITE_API_URL=http://localhost:5000/api`.
4. Run `npm run dev` to start the frontend.
